# Comp Club

Test assignment by Yadro

## Build 

```console
$ cmake -B ./build/ .
$ cd build
$ cmake --build .
```

## Run

```console
$ cd build
$ ./str/compclub <file>
```

## Test

```console
$ cd build
$ ctest
$ # OR
$ ./test/tests 
```

Some unit tests provided, same as simple input examples (see [test_files](test_files)).
